Requirements:

  • Python 3
  • Pip modules:
    • mistune
    • watchdog

Step one:

Clone the repo

git clone git@github.com:PotteriesHackspace/knowledgebase.git

cd knowledgebase/

Step two:

Run md.py

Step three:

Write your article in the md directory, your new pages will appear under the pages folder.

Submitting the changes

Add, commit, push

git add .

git commit -m "I did a thing!"

git push
